Sistema de Gestión de Historias Médicas Odontológicas

Programa de gestión clínica desarrollado en C# utilizando .NET 8, Visual Studio como entorno de desarrollo y SQLite como gestor de base de datos. Está orientado a la administración y organización de historias médicas odontológicas, permitiendo el registro completo de pacientes, la gestión de médicos y usuarios, así como el control de consultas médicas y la creación de historias clínicas digitales de forma eficiente.

Además, el sistema facilita la gestión de estudios y exámenes con archivos adjuntos, la generación de informes, constancias y récipes en PDF, el control de presupuestos y abonos, y la creación y restauración de copias de seguridad, garantizando una administración integral, segura y moderna de la información clínica.
